## 四、核心不可突破铁律

1. **DNA 主链生成后永久只读、禁止篡改**
2. **各功能分支编码规则锁定**，仅允许末尾追加字段、新增独立分支，禁止调换/删除原有字段
3. **主干、分支、分隔符、编解码任意一处擅自改动**，都会造成基因解读错位，产出错乱情绪、时空、人物记忆

---

## 五、全局唯一编解码器

### 5.1 四大固定全局接口

```typescript
// 不可新增自定义解析函数
DNA.encode(rawInput, m1BaseData, branchExts): FullDNA       // 完整编码输出
DNA.decode(dnaBinary | dnaJson): ParsedDNA                    // 全量解析
DNA.decodeBranch(dna, branchName): BranchData                  // 轻量化单分支按需读取
DNA.validate(dna): Boolean                                      // 写入/传输前强制合规校验
```

### 5.2 异常拦截分级

| 等级 | 场景 | 处理 |
|------|------|------|
| **FATAL** | 主干缺失/篡改 | 阻断记忆入库、阻断 LLM 认知组装生成 |
| **ERROR** | 分支编码错位 | 告警日志留存，丢弃错误特征不参与推演 |
| **WARN** | 版本不匹配 | 自动降级兼容解析，记录兼容告警日志 |
