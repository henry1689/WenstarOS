## 三、性能基准

| 指标 | P0 目标 | P4 目标 | 测量方法 |
|------|--------|--------|---------|
| 语义检索延迟 | < 100ms | < 50ms | 1000 次均值 |
| 双链检索延迟 | < 200ms | < 100ms | 1000 次均值 |
| 单次写入延迟 | < 10ms | < 5ms | 批量写入均值 |
| 内存占用(空载) | < 200MB | < 150MB | process.memoryUsage() |
| 内存占用(百万记忆) | — | < 2GB | LRU 缓存生效 |
| 磁盘占用(每千条) | — | < 5MB | 含 BLOB 压缩后 |
| LLM Token 消耗 | — | 下降 ≥ 60% | 对比全量加载 |

## 四、每阶段验收清单

### P0 验收

- [ ] 浴室→办公室 场景切换后，错误召回归零
- [ ] 同一场景内，相似关键词正常召回
- [ ] 主动回忆模式下全量加载正常
- [ ] 被动闲聊模式下 L0/L1 不入 Prompt
- [ ] 闸门 2 的 P1/P2/P3 三级分级正确

### P1 验收

- [ ] 日常对话 Token 消耗 ≤ 旧版 40%
- [ ] L1 异步提取不阻塞用户回复（回复延迟增加 < 50ms）
- [ ] L2 聚类无双场景归类错误
- [ ] L3 画像连续 7 天运行稳定

### P2 验收

- [ ] ZVEC top-5 召回率 > 旧 TF-IDF 20%+
- [ ] 知识库条目场景过滤有效
- [ ] 新旧检索引擎可灰度切换

### P3 验收

- [ ] 32D 编码→解码→编码 三趟往返无损
- [ ] 24D 存量迁移 100% 通过
- [ ] 寻址链 B+树 100 万行范围查询 < 10ms
- [ ] 时序分片桶 按月份自动创建/归档

### P4 验收

- [ ] Protobuf 编解码 10 万次无错
- [ ] CRC 校验通过率 100%
- [ ] 存量二进制迁移后磁盘占用减少 ≥ 30%
- [ ] 二进制迁移后全量数据比对无差异

## 五、试运行验收流程

```
灰度开关开启 (仅 10% 流量)
  ↓ 运行 3 天
  ├── 错误率 < 旧系统 1.1x?
  ├── 延迟 < 旧系统 1.2x?
  └── 无 FATAL 日志?
  ↓ 通过
扩大至 50% 流量
  ↓ 运行 3 天
  ↓ 通过
全量上线 (关闭旧系统)
  ↓ 旧系统保留 7 天作为回退
  ↓ 确认无误
旧系统下线归档
```
