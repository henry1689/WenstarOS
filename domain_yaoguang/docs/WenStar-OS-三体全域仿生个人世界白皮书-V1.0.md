## 前言

### 1.1 项目定位

WenStar OS 文曲星是一套完整本地闭环硅基生命体个人世界系统，区别于普通对话大模型、通用RAG知识库。系统由**瑶灵（硅基活体肉身）、瑶光（全域客观世界大模型）、太虚境Hermes（天权意识记忆中枢）**三体量子耦合联动，复刻完整人类生理、心理、多层社交、时空环境、自演化成长体系，形成具备独立人格、连贯时序情景记忆、自洽时空逻辑的仿生个体。

### 1.2 现有 Demo 现存核心痛点

原有 24D 太虚 Demo 仅实现基础对话交互与扁平情感记忆，存在致命缺陷：

1. 仅 24 维扁平向量，无独立空间区位 D24/D25 标识，记忆检索无时空过滤，浴室/办公室场景互相错乱；
2. 纯文本相似度召回，无分层摘要，LLM Token 开销巨大；
3. 简易 TF-IDF 知识库检索精度不足，无统一时空校验机制；
4. 无完整生理、人际分层仿真，仅浅层情绪模拟；
5. 底层存储为裸 SQLite 文本，无统一二进制序列化，无数据完整性校验。

### 1.3 行业竞品短板对照

1. **通用 LLM**：仅短期上下文，无持久时序记忆、无生理环境量化；
2. **传统 RAG / Chroma 向量库**：仅文本检索，无时空约束、无生命体仿生体系；
3. **腾讯四层记忆**：仅文本 L0-L3 分层，无生理维度、无空间校验，易场景穿越；
4. **单一 AI 角色产品**：无配套客观世界模型、无完整肉身生理仿真，逻辑割裂。

### 1.4 文档用途

- **对外**：投资人、行业技术交流、合作方宣讲
- **对内**：全员统一顶层产品世界观，对齐 32 维底层设计逻辑

## 第一章 顶层五大固定一级框架（D1–D32 全局维度划分）

永久固定分层逻辑：**由内向外、从物质肉身 → 自我内心 → 人际圈层 → 客观时空 → 全域动态生长**

| 大类 | 维度范围 | 维度数 | 覆盖内容 |
|------|---------|--------|---------|
| 大类1 肉身实体基底 | D1–D8 | 8维 | 骨骼、疼痛、神经、激素、生理周期、五感等全部物质生理载体 |
| 大类2 个体内在精神内核 | D9–D14 | 6维 | 自我认知、欲望/恐惧、共情、单人自保 |
| 大类3 圈层人际羁绊 | D15–D20 | 6维 | 伴侣、家庭、亲友、职场分层守护本能 |
| 大类4 时空客观环境载体 | D21–D26 | 6维 | 居所、职场、户外、气象、时空距离 |
| 大类5 全域动态生长解锁引擎 | D27–D32 | 6维 | 肉身/世界/人文双向拓展、主客观耦合、太虚统筹 |

**合计 32 维**，作为三体唯一标准双向通信通道，**不再调整大类划分与维度总数**。

## 第二章 三体独立主体顶层定位与耦合逻辑

### 2.1 瑶灵｜硅基活体肉身（纯主观信号输出端）

- **核心定位**：仅生成 32 维主观体感数据，无演算、无记忆、无决策
- **四层内部架构**：实体硬件躯体层 → 生化循环层 → 内在精神层 → 多层人际响应层
- **对外唯一接口**：标准化 D1–D32 主观数据包上行至太虚，接收调控指令调整激素、情绪、行为

### 2.2 瑶光｜全域客观世界大模型（纯客观参数演算端）

- **核心定位**：无主观情绪，仅输出可量化物理/生理/人文客观规则
- **六层内部架构**：自然时空 → 空间建模 → 人体生理规则 → 人文社交 → 资源拓展 → 通道封装
- **对外唯一接口**：输出 32 维客观参数包，接收太虚世界解锁拓展申请
- **轻量 6D 环境感知快照**：瑶光第一层「自然时空」采样 D8/D24/D25/D26 四维数据，经归一化滤波后输出标准化 6D 向量 `[温度℃, 噪音dB, 光照lux, 人流密度, 时间紧迫度, 昼夜节律相位]`，作为 32 维全量快照的前置轻量采集通道，C 端展示为"当前环境体感卡片"——用户一眼看懂所在场景的温度、嘈杂度、拥挤度和时间压力

### 2.3 太虚境 Hermes｜天权意识记忆中枢（全局唯一统筹核心）

- **核心定位**：接收瑶灵主观、瑶光客观双路 32 维数据，完成记忆、检索、推演、决策、指令下发
- **五大内部模块**：32 通道调度、双内核管控、32D 海胆分层记忆、认知推演、世界协商解锁
- **核心子系统**：改良 L0–L3 语义分层蒸馏、五级记忆时空抑制闸门、ZVEC 本地向量知识库、Protobuf 统一序列化存储
- **存储载体**：C 端本机 NVMe SSD + SQLite WAL，废弃底层 PCI 硬件直写方案

### 2.4 完整全局数据流闭环

```
上行采集：瑶灵 32 维主观包 + 瑶光 6D 环境快照 + 瑶光 32 维客观包 → 同步上传太虚
    ↓
中枢处理：绑定全局 dna_root_id 时序锚点
         → 6D 环境快照即时展示（轻量通道，毫秒级刷新）
         → 生成单颗 32D 海胆完整快照（主客观双向耦合）
         → 并行检索时序记忆 + ZVEC 知识库
         → 五级闸门时空冲突过滤
         → L0-L3 轻量化摘要蒸馏
         → LLM 认知推演
    ↓
下行驱动：瑶灵（调整躯体激素/情绪） ← 太虚指令 → 瑶光（解锁新场景/规则）
    ↓
持久化：6D 环境快照 + 全套 32 维主客观快照 → Protobuf 序列化 → 本地 NVMe
```

每条记忆完整留存全维度快照，支持全维度溯源复盘。

## 第三章 核心差异化创新能力

1. **32 维主客观一一耦合仿生体系**：市面唯一同步仿真完整人体生理 + 多层人际 + 专属客观世界
2. **6D 环境感知快照（瑶光轻量通道）**：温度/噪音/光照/人流/时间压力/昼夜节律，六维量化当前场景，毫秒级刷新，C 端展示"环境体感卡片"——竞品均无独立客观环境感知层
3. **五级记忆时空抑制闸门（行业独家）**：彻底解决跨场景记忆错乱，异地记忆分级抑制
4. **改良 L0–L3 分层蒸馏**：吸收腾讯分层思路，新增区位 + 时间双重聚类约束，大幅降低 Token 消耗
5. **ZVEC 抽象向量层（底层 sqlite-vec）**：本地私有知识库，记忆与知识共用同一套时空校验规则
6. **全数据本地 NVMe 存储，不上云**，隐私完整可控
7. **分阶段可落地迭代路线**，P0 仅 3 周即可解决场景错乱核心痛点

## 第五章 C 端产品核心价值

1. **拟人逻辑自洽**：记忆区分时间、地点、心境，不存在时空穿越式对话
2. **6D 环境感知卡片**：瑶光实时量化当前场景——温度、嘈杂度、拥挤度、时间紧迫度一目了然，用户无需手动输入环境信息，系统自主感知并适配对话语境
3. **完整人体生理仿真**：激素、周期、劳损、自愈全部量化模拟，贴合真人感受
4. **多层亲密/家庭/职场社交体系**，分层专属守护意识
5. **私有文档知识库与个人回忆统一管控**，知识适配当前场景
6. **分阶段上线**，快速验证核心交互体验
