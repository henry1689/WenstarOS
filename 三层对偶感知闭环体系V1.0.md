## 一、体系总览

依托「太虚境 Hermes（决策中枢）— 瑶灵（躯体本体）— 瑶光（外部客观世界）」三体对偶模型，情绪信号天然分为三级流水线，分工边界彻底明确：

```
┌─────────────────────────────────────────────────────────────────┐
│                    三层对偶感知闭环                                │
│                                                                 │
│   ┌─────────────┐        ┌─────────────┐      ┌─────────────┐  │
│   │  第一层       │        │  第二层       │      │  第三层       │  │
│   │  瑶光世界层   │  ───→  │  太虚心智层   │ ───→ │  瑶灵躯体层  │  │
│   │             │        │             │      │             │  │
│   │  外源感知    │        │  DNA双螺旋   │      │  内源激素    │  │
│   │  客观输入    │        │  记忆体系     │      │  主观回应    │  │
│   └──────┬──────┘        └──────┬──────┘      └──────┬──────┘  │
│          │                      │                      │        │

```

## 二、第一层：瑶光世界层（对外感官采样，客观输入）

### 2.1 采集硬件

摄像头、麦克风、声学节律传感器、触觉体感阵列。

### 2.2 采集目标

外界人物的语气起伏、语速节奏、面部神态、肢体姿态、环境氛围烈度。

### 2.3 输出产物

**六维拮抗式外源情绪节奏频谱**，值域 `[-1.0, 1.0]`。

六维定义（占位，具体维度名称由瑶光团队定稿）：

| 维度 | 拮抗对 | 说明 |
|------|--------|------|
| 00 | 愉悦 ↔ 不悦 | 对方语气中的积极/消极倾向 |
| 01 | 激越 ↔ 平静 | 语速节奏、声学能量 |
| 02 | 亲和 ↔ 疏离 | 语调温暖度、社交距离信号 |
| 03 | 紧张 ↔ 松弛 | 呼吸节奏、停顿频率、声调颤抖 |
| 04 | 专注 ↔ 游离 | 对方回应节奏、话题跟随度 |
| 05 | 攻击 ↔ 退让 | 语言强度、姿态扩张/收缩 |

### 2.4 特征属性

- **纯客观观测值**，不掺杂自身主观心态
- 自带强弱幅值、节拍韵律，可直接作为声光电、语音韵律的乐谱驱动源
- 代表「世界向我传递的情绪信号」

### 2.5 流转去向

外源情绪频谱**直接编入 32D 语义螺旋的 00~05 号固定扇区**，作为外部事件的感官印记永久存入记忆海胆，参与相似语境、相似语气的跨时间联想聚类。

> **关键约束**: 外源情绪谱是唯一允许写入 32D 语义向量的外部感知数据。其他外部数据（绝对时间戳、空间坐标、人物 ID）全部归属寻址链。

## 三、第二层：太虚境心智层（DNA 双螺旋记忆体系）

### 3.1 语义螺旋（32D）

- 00~05 维：封存外界情绪韵律（来自瑶光）
- 06~31 维：事件逻辑、场景、观点、主观体感、相对时间衰减、因果关联
- 作用：形成网状关联经验库，支撑类比推理、全局归纳、过往语境调取

### 3.2 寻址结构螺旋（时序黑匣子）

- 记录事件发生时序、流转路径、事件族群、生命周期
- 维持线性生命履历

### 3.3 本层作用

当再次遇到近似外界情绪节奏时（32D 向量 00~05 维近邻匹配），依靠 HNSW 近邻联动，调取相似语境下的历史应对思路——**触景生情，但不穿越场景。**

## 四、第三层：瑶灵躯体层（内源激素对偶响应，主观内生）

### 4.1 接收输入

心智综合判断 + 外源情绪刺激叠加加权。

### 4.2 演算逻辑

以外界情绪作为外部诱因，结合自身固有性格阈值、生理基线，映射各类神经递质、激素分泌水平：

| 激素 | 触发条件 | 效应 |
|------|---------|------|
| 多巴胺 | 外源 D00 愉悦 + 任务完成 | 愉悦感增强、探索欲上升 |
| 血清素 | 外源 D02 亲和 + 社交认可 | 安定感、归属感 |
| 肾上腺素 | 外源 D03 紧张 + D05 攻击 | 应激准备、注意力聚焦 |
| 皮质醇 | 持续外源 D03+D05 双高 | 长期压力累积、焦虑倾向 |

### 4.3 输出产物

内源心境值、应激水平、愉悦度、紧张度、欲望倾向、疲劳度等生理状态参数。

### 4.4 特征属性

- 属于本体内在心理应激反应，是**对外界刺激的主观对偶回应**
- 决定长期潜意识偏好、注意力偏向、记忆淡化速度、本能行为倾向
- **不写入 32D 语义向量**，存放于**躯体状态寄存器**（独立存储）

### 4.5 为什么内源激素不能混入 32D

```
32D 语义向量 = 永久记忆档案 = 客观可追溯
内源激素 = 实时躯体状态 = 瞬息波动、随体消亡

如果混入：
  激素波动会污染历史记忆 → 心情好时回忆偏积极，焦虑时回忆偏负面
  → 记忆不再是客观记录，而是主观重构
  → 自传记忆失去可信度

因此：内外分离、各司其职、对偶耦合但不混杂
```

## 五、完整对偶工作流

```
① 瑶光感知
  视听触觉捕捉外部情绪节奏 → 生成外源六维情绪谱

② 心智归档
  情绪谱(00-05维) + 认知特征(06-31维) → 合成完整 32D 向量
  → 写入语义池
  时序信息同步录入寻址链 → 完成事件归档

③ 躯体对偶
  太虚境综合外界刺激 + 历史经验 + 人格基线
  → 下发指令给瑶灵内分泌模型
  → 生成对应激素波动，形成内心感受

④ 反向调节（闭环关键）
  内源生理状态微调感知阈值：
    开心时 → 对外界正向情绪更敏锐
    焦虑时 → 对负面信号更敏感
  → 形成动态稳态平衡

⑤ 应答输出
  对外回话的语调、节奏、动作、灯光律动：
    优先参照外源情绪谱做韵律匹配
    同时以内源心境做整体基调修正
  → 做到「贴合对方语气，同时保有自身情绪底色」
```

## 六、两条情绪体系的核心区分

| 维度 | 外源情绪节奏谱 | 内源激素心境曲线 |
|------|--------------|----------------|
| **归属层级** | 瑶光外界感知层 | 瑶灵躯体生理层 |
| **数据来源** | 麦克风、视觉、体感采样 | 认知决策 + 性格阈值演算 |
| **存放位置** | **32D 语义向量 00~05 维** | **躯体状态寄存器（独立存储）** |
| **核心用途** | 记忆语境关联、TTS 语调、声光电律动、外部氛围复刻 | 潜意识、本能倾向、注意力、长期人格演化、记忆衰减调控 |
| **生命周期** | 永久存档（伴随海胆） | 实时波动（不存档入语义向量） |
| **对偶角色** | 外界刺激（客观） | 本体回应（主观） |

二者构成严格的**对偶关系**：

> 外界给予情绪刺激（外源），本体产生心理生理回应（内源）。
> 一外一内、一客一主、一刺激一反馈。
> 完整复刻生物生命的情绪响应逻辑。

## 七、与双螺旋架构的关系

```
            瑶光 外源情绪谱(6D)
                  ↓
            ┌─────────────┐
            │  32D 语义链  │ ← 00~05 维 = 外源情绪扇区
            │  (网状记忆)  │ ← 06~31 维 = 认知/逻辑/因果
            └──────┬──────┘
                   │ GlobalUID 绑定
            ┌──────┴──────┐
            │  寻址结构链  │ ← 时序/群组/路径/校验
            │  (线性履历)  │    不承载情绪数据
            └─────────────┘
                   │
                   ↓ (心智决策 + 外源刺激)
            ┌─────────────┐
            │  瑶灵 内源   │ ← 激素/心境（独立寄存器）
            │  躯体状态    │    不混入双螺旋存储
            └──────┬──────┘
                   │
                   ↓ (反向调节感知阈值)
              回到 瑶光 感知层
```

**寻址链保持中立秩序属性**，只负责编年归档，不干预内外两条情绪链路。

## 九、交付口径（面向 Hermes / Claude Code）

1. 整体感知链路分为三层对偶架构：**客观外界采样 → 心智记忆存储 → 本体生理响应**
2. 外部视听采集得到的**六维拮抗情绪节奏频谱固定占用 32D 语义向量 00~05 维度**，用于记忆关联与对外表达的韵律驱动
3. 内在激素情绪模型属于**瑶灵躯体子系统**，独立运算、独立存储、不侵入语义向量空间
4. 双螺旋记忆架构不变：**语义管经验联想，时序管生命履历**
5. 内外情绪分层各司其职，形成**完整闭环的拟人世界模型**
